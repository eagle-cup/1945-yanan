# Design Context

## Users
- **目标用户**：大学生和青年参观者
- **使用场景**：站在实体沙盘前，用手机扫描NFC/二维码后进入网页，竖屏单手浏览
- **用户任务**：快速了解每个延安历史地点的核心事件、观看视频、认识历史人物
- **情感目标**：让年轻一代感受到1945年延安的历史厚重感、革命先辈的精神力量，产生"课本之外"的鲜活历史体验

## Brand Personality
- **三个关键词**：庄重（Solemn）、温暖（Warm）、力量感（Powerful）
- **品牌声音**：像一个有学养的历史讲述者——不喊口号、不刻板说教，而是娓娓道来，用真实的历史细节打动人
- **情感基调**：肃然起敬但不沉重，有历史温度但不轻浮

## Aesthetic Direction
- **主题**：深色/暗红色调为主，金色和暖光点缀，营造"窑洞里的灯光"氛围
- **参考意象**：延安窑洞夜晚透出的暖光、旧报纸的质感、老照片的颗粒感、红军军装的粗布纹理
- **队徽配色提取**：
  - 底色：深藏蓝/午夜黑
  - 金色：暖金色（月亮和星星）
  - 红色：正红（飘带）
- **明确不要**：花哨的渐变、玻璃态/毛玻璃效果、荧光霓虹色、过于现代简约的"苹果风"、AI生成的廉价装饰

## Design Principles
1. **暗中有光**：深色背景为主，但关键信息用暖金色或柔白光"点亮"，模拟窑洞里看史料的感觉
2. **少即是多**：移动端屏幕小，每页只传达一个核心信息，避免堆砌
3. **质感优先**：使用纸张纹理、老照片、布纹等材质感元素，让数字界面有"触摸历史"的质感
4. **字体有骨**：标题用有历史感的宋体/刻本字体，正文保持清晰易读
5. **交互有温度**：过渡动画像翻书页一样自然，不做炫技式动效

## Technical Constraints
- 移动端优先（375-414px基准宽度）
- 纯静态HTML/CSS/JS，无需构建工具
- 部署到GitHub Pages
- 视频和音频使用HTML5原生标签
