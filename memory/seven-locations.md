# 七个地点详细信息

## 1. 整风运动 — 杨家岭中央党校
- **时间**：1941年5月 - 1945年4月（1945年初完成）
- **历史意义**：全党思想统一，为中共七大召开奠定思想基础
- **核心内容**：反对主观主义、宗派主义、党八股，确立"实事求是"思想路线
- **重要人物**：毛泽东（《改造我们的学习》《整顿党的作风》）、刘少奇（《论共产党员的修养》）
- **设计关键词**：学习、整风文件、党校课堂、思想改造
- **代表颜色**：深棕色（书籍、学习）

## 2. 中共七大 — 杨家岭中央大礼堂
- **时间**：1945年4月-6月
- **历史意义**：确立毛泽东思想为全党指导思想，选举产生"五大书记"
- **核心内容**：《论联合政府》、确立毛泽东思想、修改党章
- **重要人物**：毛泽东、朱德、刘少奇、周恩来、任弼时（五大书记）
- **设计关键词**：礼堂、红旗、团结、胜利
- **代表颜色**：正红色（党的象征）

## 3. 窑洞对 — 杨家岭毛泽东旧居
- **时间**：1945年7月
- **历史意义**：黄炎培提出"历史周期率"问题，毛泽东给出"民主"答案
- **核心内容**："其兴也勃焉，其亡也忽焉" → "只有让人民来监督政府"
- **重要人物**：毛泽东、黄炎培
- **设计关键词**：窑洞、对话、民主、历史思考
- **代表颜色**：土黄色（窑洞、黄土高原）

## 4. 抗大 — 抗日军政大学旧址
- **时间**：1936年建校，1945年全年办学
- **历史意义**：培养抗战干部，全年持续输送人才
- **核心内容**："团结、紧张、严肃、活泼"校训
- **重要人物**：林彪（校长）、徐向前（代校长）、学员中的开国将领
- **设计关键词**：军校、学习、训练、青年
- **代表颜色**：军绿色（军事院校）

## 5. 抗战胜利集会 — 革命纪念馆
- **时间**：1945年8月（日本投降）
- **历史意义**：庆祝十四年抗战胜利，延安全城欢庆
- **核心内容**：9月5日南门外广场万人集会庆祝
- **重要人物**：朱德（在庆祝大会上讲话）、毛泽东（《庆祝抗战最后胜利》）
- **设计关键词**：胜利、欢庆、红旗、人民
- **代表颜色**：金红色（胜利的喜悦）

## 6. 重庆谈判筹备 — 枣园领导人旧居
- **时间**：1945年8月-10月
- **历史意义**：党中央在枣园密集开会，筹备重庆谈判
- **核心内容**：制定谈判方针、《对目前时局的宣言》、毛泽东赴重庆
- **重要人物**：毛泽东、周恩来、王若飞
- **设计关键词**：决策、谈判、和平、智慧
- **代表颜色**：深蓝色（沉着、智慧）

## 7. 战后西北统战治理 — 花石砭中共中央西北局旧址
- **时间**：1945年10月起
- **历史意义**：习仲勋主持西北局，统筹战后边区治理与统一战线
- **核心内容**：边区经济恢复、统一战线工作、少数民族工作
- **重要人物**：习仲勋（西北局书记）
- **设计关键词**：治理、建设、团结、西北
- **代表颜色**：橙黄色（西北大地、建设）

## 每个地点的人物卡片设计规范
- 每页至少2-3个核心人物
- 人物卡片：头像 + 姓名 + 一句话简介
- 点击展开：详细生平 + 历史贡献 + 原声音频（可播放语录）
- 图片来源：历史照片或绘制插图
