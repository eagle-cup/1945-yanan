# 项目概述：1945延安 · 沙盘互动网页群

## 项目背景
这是一个以1945年延安为主题的创新创业项目。项目主体是一个按延安地理位置设计的实体沙盘，沙盘上有7个地点标记，每个点位放置NFC标签或二维码，参观者通过手机扫码进入对应的手机网页。

## 项目目标
制作一个网页群，包含：
- **7个独立主题网页**（对应7个地点）
- **1个统领大网页**（地图式导航 + 答题系统）
- **1个总体入口网页**（统一整合）

## 技术方案
- 纯静态HTML/CSS/JS网页（适合手机浏览）
- 响应式设计（移动端优先）
- 可部署在GitHub Pages或类似静态托管服务

## 7个地点（按时间逻辑排序）
1. **整风运动** — 杨家岭中央党校（年初，思想统一）
2. **中共七大** — 杨家岭中央大礼堂（春夏，确立路线）
3. **窑洞对** — 杨家岭毛泽东旧居（七月，黄炎培到访）
4. **抗大** — 抗日军政大学旧址（全年，培养干部）
5. **抗战胜利集会** — 革命纪念馆（八月，日本投降）
6. **重庆谈判筹备** — 枣园领导人旧居（胜利后，筹备谈判）
7. **战后西北统战治理** — 花石砭中共中央西北局旧址（十月，习仲勋主持）

## 每个子网页的结构（三页式）
1. **封面页**：队徽 + 地点特色 + 美观设计
2. **视频页**：自制历史介绍视频
3. **人物卡片页**：重要人物卡片（可点击进入详情），包含：
   - 人物图片
   - 人物原声语录（点击播放音频）
   - 人物详细介绍

## 大网页功能
- 地图式界面，7个地点串联
- 点击地点切入对应子网页
- 答题系统入口：
  - 题库功能
  - 答题功能
  - 错题本功能
