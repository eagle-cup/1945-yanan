# 答题系统设计

## 功能需求
1. **题库功能**：存储所有题目，按地点分类
2. **答题功能**：用户选择答案，实时反馈对错
3. **评分功能**：答题结束后显示总分
4. **错题本功能**：自动收集错题，支持复习

## 题库结构（JSON格式）
```javascript
const questionBank = {
  "zhengfeng": [  // 整风运动
    {
      id: "zf001",
      question: "延安整风运动的主要方针是什么？",
      options: [
        "A. 惩前毖后，治病救人",
        "B. 残酷斗争，无情打击",
        "C. 坦白从宽，抗拒从严",
        "D. 团结—批评—团结"
      ],
      answer: 0,  // 正确答案索引
      explanation: "整风运动实行'惩前毖后，治病救人'的方针..."
    }
  ],
  // ... 其他地点
};
```

## 每个地点的题目数量
- 每个地点至少5道题目
- 总共至少35道题目
- 题型：单选题（可扩展多选题）

## 答题模式
1. **地点模式**：选择某个地点进行专项答题
2. **综合模式**：从所有题库随机抽取（如10-20题）
3. **挑战模式**：限时答题

## 评分规则
- 每题10分（可根据题目数量调整）
- 总分 = 正确题目数 × 每题分数
- 等级评定：优秀(90%+)、良好(70%+)、及格(60%+)、不及格

## 错题本
- 使用localStorage存储
- 存储结构：
```javascript
{
  wrongQuestions: [
    { questionId, userAnswer, timestamp }
  ],
  reviewHistory: [
    { questionId, reviewCount, lastReviewDate, mastered }
  ]
}
```
- 功能：
  - 查看所有错题
  - 重新答题（仅错题）
  - 标记"已掌握"（从错题本移除）
  - 导出/清空错题本

## UI设计
- 答题页面：题目 + 四个选项按钮
- 进度条显示答题进度
- 正确/错误反馈动画（绿色/红色闪烁）
- 答案解析展示
- 结果页：分数 + 等级 + 错题数量 + 复习错题按钮
