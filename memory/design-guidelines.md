# 设计规范

## 色彩体系

### 主色调
- **革命红**：#C41E24 — 主要按钮、标题、强调元素
- **深红**：#8B0000 — 背景渐变、重要区域边框

### 辅助色
- **黄土金**：#D4A574 — 延安黄土高原的地域特色
- **窑洞棕**：#8B7355 — 窑洞建筑的质感

### 地点专属色（7色体系）
| 地点 | 色号 | 用途 |
|------|------|------|
| 整风运动 | #6B4226 | 书卷棕 |
| 中共七大 | #C41E24 | 正统红 |
| 窑洞对 | #B8956A | 窑洞黄 |
| 抗大 | #4A6741 | 军旅绿 |
| 抗战胜利集会 | #DAA520 | 胜利金 |
| 重庆谈判筹备 | #2C3E6B | 沉着蓝 |
| 战后西北统战 | #E07B3C | 西北橙 |

### 中性色
- 背景：暖白 #FFF8F0
- 文字：深灰 #2C2C2C
- 次文字：中灰 #666666

## 字体系统
- **标题字体**：使用思源宋体（Source Han Serif）或方正清刻本悦宋（体现历史感），备用：STSong, serif
- **正文字体**：思源黑体（Source Han Sans），备用：PingFang SC, Microsoft YaHei
- **英文/数字**：使用与中文匹配的衬线体

## 页面设计原则
1. **移动优先**：375px-414px 为基准设计宽度
2. **竖屏浏览**：适应手机竖屏习惯
3. **大触控区域**：按钮最小44×44px
4. **简洁优雅**：避免过多装饰，突出历史庄重感
5. **氛围营造**：使用纹理背景（纸质、布料质感）

## 封面页设计要素
- 队徽居中或左上角
- 地点名称大字标题
- 地点特征插画/历史照片
- 向下滑动引导箭头
- 日期标注（历史时间）

## 人物卡片UI
- 圆形头像框
- 姓名 + 职务
- 一句话金句预览
- 点击展开动画（卡片翻转或底部弹出）
- 详情页：大图 + 生平 + 语录（带播放按钮）

## 大网页地图设计
- 手绘风格或复古地图风格
- 7个地点用不同颜色标记
- 地点之间用虚线或路线连接
- 点击地点 → 缩放动画 → 跳转子网页
- 地图可拖拽/缩放（移动端手势支持）
